# Ride Sharing System (Smalltalk + C++)

This project demonstrates core object-oriented programming principles in two languages:
- **C++17** (class-based OOP)
- **Smalltalk** (pure OO environment)

## Structure
- `cpp/` — C++ implementation (CMake project)
- `smalltalk/` — Smalltalk implementation (fileout)
